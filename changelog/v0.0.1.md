# Changelog

## [0.0.1] - 2026-04-14

### 🎉 初始发布

女娲 Runtime OS 核心架构完成，包含 6 层模块和端到端集成 Demo。

### 新增

#### 核心模块
- **@nuwa-os/event-bus** — 发布-订阅事件总线
  - Topic 路由，支持 `*`（单级）和 `**`（多级）通配符
  - 事件优先级（HIGH / NORMAL / LOW）
  - 事件历史记录（可配置容量）
  - 49 tests

- **@nuwa-os/kernel-core** — 微内核
  - 可配置 tick 间隔的事件循环（EventLoop）
  - 优先级进程调度器（Scheduler）
  - 系统时钟（SystemClock），支持时间缩放和暂停
  - 基础进程抽象（BaseProcess）
  - 55 tests

- **@nuwa-os/sensor-hal** — 传感器硬件抽象层
  - 统一传感器接口（ISensor<T>）
  - 传感器注册表（SensorRegistry）
  - 虚拟摄像头（VirtualCamera）：640x480 RGB 帧，可配置帧率
  - 虚拟温度计（VirtualThermometer）：18-30°C 区间，含湿度
  - 传感器数据 → NuwaEvent 转换工具
  - 53 tests

- **@nuwa-os/agent-runtime** — Agent 运行时框架
  - 感知-思考-行动（Perceive-Think-Act）循环
  - Agent 上下文（AgentContext）：事件订阅、发布、日志
  - Agent → 内核进程适配器（AgentProcess）
  - Agent 运行时管理器（AgentRuntime）
  - 可替换的 `think()` 方法插槽
  - 45 tests

- **@nuwa-os/device-gateway** — 设备接入网关
  - 统一设备接口（IDevice），命令/数据双向通信
  - 设备注册表（DeviceRegistry）
  - 虚拟摄像头设备（VirtualCameraDevice）：帧捕获、分辨率设置
  - 虚拟温度计设备（VirtualThermometerDevice）：温度读取、单位切换
  - 设备数据 → Event Bus 桥接
  - 网关 → 内核进程适配器（GatewayProcess）
  - 62 tests

#### 集成 Demo
- **@nuwa-os/demo-vision-agent** — 端到端视觉感知 Demo
  - VisionAgent：订阅摄像头数据，处理视觉帧，输出分析动作
  - NuwaOS 一键启动类：boot() → 全栈初始化
  - 实时数据流可视化 Demo（彩色终端输出）
  - 完整数据通路验证：VirtualCamera → EventBus → VisionAgent → actions
  - 29 tests

### 技术细节
- TypeScript 5.4+ strict mode，全量零编译错误
- Vitest 测试框架，`--pool=forks` 模式
- npm workspaces monorepo，零外部运行时依赖
- 每个模块类型自包含，无跨包依赖
